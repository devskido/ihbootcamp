Day 09 - Requirements

과제 개요

이번 수업에서 JavaScript의 변수 선언(var, let, const), 기본 자료형(number, string, boolean, null, undefined, Symbol, BigInt), typeof 연산자, 이스케이프 문자, 문자열 연결 연산자, 템플릿 문자열, 배열 및 객체 리터럴을 배웠습니다. 이번 과제는 HTML 없이 순수 JavaScript를 사용하여 브라우저 콘솔에서 실행되는 개인 프로필 생성기를 제작하는 것입니다. 기본 과제는 변수, 문자열, 배열, 객체를 활용해 개인 프로필 정보를 출력하며, 도전 과제는 null과 undefined 구분, 추가 데이터 처리, 템플릿 문자열 활용을 연습합니다.

기본 과제 요구사항

  1. JavaScript 파일 (profile.js)
    • 브라우저 콘솔에서 실행 가능한 JavaScript 파일을 작성하세요.
    • 기능:
      • 변수 선언:
        • var, let, const를 각각 사용하여 최소 3개의 변수를 선언.
        • 각 변수에 서로 다른 데이터 타입(number, string, boolean, null, undefined, Symbol, BigInt 중 선택) 할당.
      • 문자열 처리:
        • 이스케이프 문자(\n, \t)를 사용하여 프로필 정보에 줄바꿈 또는 탭 포함.
        • 문자열 연결 연산자(+)를 사용하여 최소 하나의 프로필 문장 생성.
      • 배열 리터럴:
        • 배열을 선언하여 최소 3개의 취미(hobbies)를 저장(예: ["reading", "gaming", "coding"]).
        • 배열 요소를 문자열로 결합하여 출력(예: My hobbies: reading, gaming, coding).
      • 객체 리터럴:
        • 객체를 선언하여 개인 정보 저장(예: { name: "John", age: 20, isStudent: true }).
        • 객체 속성을 사용하여 최소 하나의 프로필 문장 생성.
      • typeof:
        • 최소 2개의 변수(배열, 객체 포함)에 대해 typeof를 사용하여 데이터 타입 출력.
    • 요구사항:
      • var, let, const 각각 최소 1회 사용.
      • 최소 3가지 데이터 타입(number, string, boolean, null, undefined, Symbol, BigInt 중 선택) 사용.
      • 이스케이프 문자(\n, \t)를 최소 1회 사용.
      • 문자열 연결(+)로 최소 1개의 문장 생성.
      • 배열에 최소 3개의 요소, 객체에 최소 3개의 속성 포함.
      • typeof로 최소 2개의 변수 타입 출력.
  2. 실행 방법
    • JavaScript 파일을 작성 후, 간단한 HTML 파일에 <script src="profile.js"></script>를 추가하여 실행:
    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Profile Generator</title>
</head>
<body>
  <script src="profile.js"></script>
</body>
</html>
    • HTML은 제출에 포함하지 않아도 됩니다.
  3. 파일 구조
    • 프로젝트 폴더를 구성:
    profile-generator/
├── profile.js

도전 과제 (선택)

기본 과제를 완료한 학생들은 null과 undefined 구분, 템플릿 문자열, 추가 데이터 처리를 연습하세요.

  1. 추가 기능 구현
    • null과 undefined 구분:
      • null과 undefined를 각각 변수에 할당.
      • typeof로 두 변수의 타입을 출력하고, === 연산자로 두 값이 다른지 비교(예: console.log(null === undefined); // false).
    • 템플릿 문자열:
      • 템플릿 문자열(``)을 사용하여 프로필 정보를 출력(예: `Name: ${name}, Age: ${age}`).
    • 추가 데이터 처리:
      • 배열에 null 또는 undefined를 포함하고, 각 요소의 데이터 타입을 출력.
      • 객체에 새로운 속성(예: email: null)을 추가하고, 해당 속성값이 null인지 확인 후 출력.
  2. 도전 과제 파일 구조
    • 동일한 profile.js에 추가 기능을 포함.

주의사항

  • JavaScript 문법: var, let, const, 기본 자료형(number, string, boolean, null, undefined, Symbol, BigInt), typeof, 이스케이프 문자(\n, \t), 문자열 연결(+), 템플릿 문자열, 배열/객체 리터럴만 사용하세요. 루프, 함수, DOM 조작은 사용하지 마세요.
  • 실행 환경: 브라우저 콘솔(F12)에서 실행하거나, 간단한 HTML 파일로 테스트하세요. Node.js는 사용하지 않습니다.
  • 코드 가독성: JavaScript 코드는 들여쓰기를 통해 가독성을 높이고, 주석(//)으로 주요 코드(변수 선언, 문자열 처리, typeof 사용 등)를 설명하세요.
  • 테스트: 브라우저 콘솔에서 profile.js를 실행하여 변수, 배열, 객체, 데이터 타입이 올바르게 출력되는지 확인하세요.
  • 출력 형식: 출력은 읽기 쉬운 형식으로(예: Variable myAge is: number, My hobbies: reading, gaming, coding).
